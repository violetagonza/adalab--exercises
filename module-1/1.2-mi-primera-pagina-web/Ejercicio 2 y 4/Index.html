HTML, CSS y Javascript: elementos para la construcción de una página web

HTML

El HTML es el lenguaje de programación web que sirve para dar estructura y contenido a las páginas, que además ayuda a que los usuarios con problemas de visión puedan entender las jerarquías dentro de loe elementos de la página.

CSS

CSS son las hojas de estilo. Con ese lenguaje lo que conseguimos es dar estilo a las páginas, en el sentido de diseño, colores, formas, etc. Las hojas de estilo van enlazadas a los documentos html.

Javascript

Javascript, por su parte, es el lenguaje de programación que sirve para dar interactividad a las páginas webs. De los tres, es el lenguaje que más cuesta a la hora de aprender a programar front end, pero una vez aprendido, puede reportar muchas satisfacciones al desarrollador.

Agile, el método de trabajo en equipo que está revolucionando las empresas tecnológicas y de desarrollo de software.

Adalab, el curso intensivo para programación front-end que ayuda a las mujeres a incorporarse al mundo de la tecnología.
